Обработка "Поиск электронных писем по объектам"

Обработка доступна только при использовании встроенного почтового клиента и предназначена для поиска электронных писем по объектам конфигурации (Контрагнеты, контактные лица, физические лица и пр.).

В самих документах "Электронное письмо" храняться только адреса электронной почты отправителей и получателей писем. Данная обработка определяет список адресов электронной почты тех объектов, по которым необходимо показать историю электронной переписки и выстраивает список электронных писем по собранным адресам электронной почты.

Пользователь указывает вид сравнения для отбора, и значения отбора (элементы справочников). Список электронных писем обновляется автоматически при изменении вида сравнения или значения отбора.

Данная обработка вызывается из форм элементов и списков справочников "Контрагенты" и "Контактные лица" с предустановленными значениями отборов.
