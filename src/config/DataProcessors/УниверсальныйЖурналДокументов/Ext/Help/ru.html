Обработка "Универсальный журнал документов".

 

Данная обработка позволяет выводить список документов, вместе со значениями реквизитов.

При этом состав списка документов и реквизитов может произвольно настраиваться.

 

Работа со списком документов.

Методы работы с сформированным списком документов полностью аналогичны работе с формами списка документов, или журналами.

При этом доступно:

  • создание, копирование, изменение документа
  • проведение / отмена проведения текущего документа
  • установка / снятие пометки удаления текущего документа
  • установка периода отбора документов по дате
  • поиск документа по номеру
  • ввод документов на основании текущего
  • просмотр движений документа по регистрам
  • формирование отчета "Движения документа по регистрам", вывод структуры подчиненности документа
  • использование аналога системного сервиса "Вывести список"

Обновление списка документов выполняется нажатием кнопки  на командной панели формы.

Следует учитывать, что при первоначальном открытии обработки, список документов не заполняется. Данное поведение обусловлено тем, что по умолчанию в настройках обработки включен режим вывода всех существующих в базе данных документов, без ограничения по дате и виду документа. Что может занимать значительно время. Для вывода этого списка, необходимо либо произвести первоначальную настройку обработки (форма настройки вызывается нажатием кнопки "Настройка"), либо нажать кнопку , что вызовет формирование списка без каких-либо ограничений. Также обновление списка происходит при установке значения периода отбора документов по дате.

Настройка обработки.

Вызов формы настройки выполняется нажатием кнопки "Настройка" командной панели обработки.

В данной форме доступны следующие настройки:

  • настройка списка видов документов, выводимых обработкой - включение вида документа в список осуществляется установкой соответствующего флажка в списке "Выводимые документы". Соответственно для исключения вида документов из списка, данный флажок необходимо снять.
  • настройка перечня дополнительных граф, выводимых обработкой. Количество дополнительных граф списка документов является неограниченным. Добавление или удаление графы выполняется в разделе "Список граф". Имя графы может быть произвольным (впоследствии показывается в заголовке соответствующей колонки списка). Каждая графа может быть связана с произвольным реквизитом для каждого вида документов, из числа указанных в списке "Выводимые документы".  При этом для одной и той же графы возможно указание реквизитов несовпадающих типов. Привязка графы к реквизиту выполняется установкой соответствующего флажка в формируемом дереве граф в правой нижней части формы.

  • установка отбора документов по значению дополнительных граф. Для задействования отбора необходимо установить флажок в колонке "Отбор" в строке соответствующей графы. Значения отбора задается в появляющейся при этом колонке "Значение". Отбор осуществляется путем сравнения значения, соответствующих для данной графы реквизитов обрабатываемых документов. Допускается указание списка возможных значений отбора

Следует учитывать, что при сравнении в процессе отборе значения отличающихся типов считаются как несоответствующие условиям отбора.. То есть, если в качестве данных для графы указаны реквизиты документов, имеющих различный тип, и при этом установлен отбор на определенное значение графы, документы с несовпадающим типом реквизита будут пропущены. Например, пусть добавлена графа, связанная с реквизитом "Контрагент" типа справочник "Контрагенты" одного документа, и реквизитом "Организация" типа справочник "Организации" другого документа. Тогда если в качестве значения отбора указать любое значение типа справочник "Контрагенты", все документы второго вида при этом через отбор не пройдут. Чтобы избежать такого поведения значение отбора следует задавать списком.

Возможно сохранение произведенных настроек, для их последующего использования. Для этого необходимо в поле "Сохр. настройки" указать имя настройки, после чего нажать кнопку "Сохранить".

Выбор сохраненной настройки для повторного использования выполняется из выпадающего списка поля "Сохр. настройки". Удаление настройки из списка сохраненных выполняется с помощью кнопки очисти значения поля "Сохр. настройки".

 

При нажатии кнопки "ОК" выполняется формирование списка документов в соответствии с указанными при этом настройками.
