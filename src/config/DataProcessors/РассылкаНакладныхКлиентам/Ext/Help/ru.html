Обработка    "Рассылка документов Покупателям (по эл. почте)"

Данная обработка предназначена для отправки эл.копий печатных документов
"Товарная накладная"  или "Счет на оплату" на электронные адреса Покупателей

За выбранный период - по документам "Реализация товаров и услуг",  
с учетом отборов по Сотрудникам, Ответственных за договор с Покупателем,
а также с учетом отборов по договорам БезНал. оплаты  (или по Всем реализациям Нал/Безнал)
Формируются электронные письма - отдельно каждому Покупателю
с вложенными Excel - файлами: "Товарная накладная"  и / или   "Счет на оплату"
имя файл содержит номер документа "Реализация товаров и услуг".

Письмо с вложенными файлами отправляется на E-mail Покупателя (при его наличии)
с эл.адреса "основная учетная запись эл.почты" текущего пользователя
или с общего адреса организации (ЗАО ТК "Яршинторг" no-reply@yst76.ru)

Текст письма
   Текст письма - стандартный, но его можно менять прямо в форме обработки.
* при нажатии на кнопку "выбор" - текста письма заполняется "стандартным текстом".

Сохранение настройки
  Сохранение настройки обработки - позволяем сохранить все видимые элементы обработки:
список выбранных Контрагентов, Реализаций, список Сотрудников, текст письма и т.д.
При открытии сохраненной настройки - можно увидеть кому и за какой период уже
были отправлены письма, а кому - нет...

Место хранения временных файлов

Перед отправкой файлов - все файлы по выбранным Покупателям сохраняются
в каталог временных файлов, который можно выбрать в форме обработки. 

* при "очистке" либо неправильном написании каталога временных файлов
   файлы сохраняются в "стандартный" каталог временных файлов.

* после отправки писем - список файлов можно просмотреть
   нажатием кнопкой "Открыть" в поле каталог временных файлов.

Периодически - эти файлы требуется удалять для освобождения места на компьютере.

Ограничения

При наличии ограничений просмотра "Заказов покупателей" у текущего пользователя -
доступна отправка писем только "своим" Покупателям, т.е. 
(включен отбор по Текущему Пользователю, Ответственному за договор с Покупателем).
