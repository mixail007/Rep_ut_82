Обработка «Журнал документов: Заказы Покупателей»

Обработка предназначена для отображения списка документов "Заказ Покупателя" и работы с документами.
Обновление списка документов выполняется нажатием кнопки

Период журнала:

При первоначальном открытии журнала, список документов заполняется автоматически за период
с 1-го числа предыдущего месяца - по текущий день. Период можно изменить нажатием кнопки  
При наличие  сохраненной настройки  у которой стоит флажок "Использовать при открытии"
- список отбирается по данной настройке.

Список документов выводится с учетом Ограничения списка документов, см. далее.

Оформления списка:

При оформлении списка документов используется изменение Цвета текста, цвета фона и шрифта в зависимости от Заказа.

  Текст "Жирный"  - означает, что данный заказ - Проверен.

Цвет текста - Синий   -  означает то, что по Заказу - есть Продажи,
т.е. на основании Заказа введен документ "Реализация Товаров и Услуг". 

Цвет текста - Красный -  означает, что Продано БОЛЬШЕ чем Количество в Заказе.
Возможно, существуют документы Продаж, с "лишним" количеством
или неправильно сделана Коррекция заказа.
Список документов можно увидеть по последней кнопке - "Структура подчиненности".

ЦветФона - Светло-Серый  - Заказ ЗАКРЫТ полностью. (Заказ полностью Отгружен и/или Откорректирован).

 

ЦветФона - Светло-Розовый - означает, что Заказ - ПЕРЕ-отгружен!  ( Отгружено больше чем Количество в Заказе !)
Возможно, существуют документы Продаж, с "лишним" количеством
или неправильно сделана Коррекция заказа.

Список документов можно увидеть по кнопке - "Структура подчиненности".

Настройка отбора:

Отбора выполняется нажатием кнопки «Отбор» командной панели обработки.
Реализация отбора представляет пользователю больше возможностей чем стандартный отбор.
Теперь можно указать в качестве отбора свойства полей, например Контрагент.Покупатель = Истина.

Кроме этого при нажатии правой кнопки мыши - можно осуществить  «Отбор по значению в текущей колонке»

Отмена Отбора выполняется нажатием кнопки «Отключить Отбор».

Отбор работает с учетом Прав и Настроек Пользователя.

 

Поиск по Номеру:

Поиск документов по номеру выполняется нажатием кнопки
При включенном режиме "Поиск по подстроке" - осуществляется поиск по введенному номеру,
при этом можно не указывать начальные символы "ТК00", а сразу вводить номер, например "14081"

 

Ограничение списка документов:

Состав документов ограничивается в зависимости от "Прав пользователя" и "Настроек Пользователя",
а также в зависимости от периода и выбранных отборов, определенных Пользователем.

Все пользователи делятся на 3 категории:

1) Видят только свои заказы

2) Видят заказы своей группы (например, Мелкий Опт)

3) Видят все заказы

Обновление списка документов выполняется нажатием кнопки

Работа со списком документов

Методы работы с сформированным списком документов практически аналогичны работе с формами списка документов, или журналами.

При этом доступно:

  •  создание,  копирование,  изменение документа
  •  проведение /  отмена проведения текущего документа
  •  удаление / отмена удаления текущего документа
  • установка отбора   по дате или  Отбора по любым графам
  • установка интеракивного отбора  "Отбор по значению в текущей колонке"
  • Отмена всех отборов  (но с учетом ограничений, см. выше)
  • Сортировка по алфавиту  и против алфавита  по текущей колонке
  • ввод документов  "на основании" текущего документа
  • формирование отчета  «История документа»,
  • вывод  структуры подчиненности документа

Обновление списка документов выполняется нажатием кнопки  "Обновить" на командной панели формы.

Общие настройки журнала

Данная обработка позволяет выводить список документов, при этом состав списка документов и реквизитов может произвольно настраиваться.

Сохранение настроек состава, отбора и позиций колонок.

Для сохранений настроек состава журнала, отбора и настроек колонок  списка документов (ширина колонок, видимость, позиция) используется стандартный механизм сохранения  и последующего восстановления настроек.
