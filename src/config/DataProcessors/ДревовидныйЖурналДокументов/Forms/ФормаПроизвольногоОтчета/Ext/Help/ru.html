Форма настройки произвольного отчёта по универсальному журналу документов на основе схемы компоновки данных

В форме доступны все стандартные возможности настройки вывода схемы компоновки данных. Вывод отчёта выполняется по кнопке "Сформировать". Настройки схемы компоновки сохраняются только в файл по кнопке с дискетой, расположенной на правой верхней панели закладки настроек.

Будьте внимательны! При восстановлении настроек из файла не проверяется соответствие восстанавливаемой настройки текущим настройкам журнала!
