Набор параметров, который использовался при формировании какого-либо отчета, можно сохранить, и затем использовать при последующих построениях этого же отчета.

Для восстановления одной из ранее сохраненных настроек используется пункт «Действия - Сохранить значения...» или соответствующая кнопка командной панели. На экран будет выдан список сохраненных наборов параметров.
 
В этом списке необходимо выбрать имя одного из ранее сохраненных наборов параметров. После нажатия кнопки «OK» список будет закрыт, а реквизиты формы будут заполнены восстановленными значениями.
