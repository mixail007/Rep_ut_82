Обработка "Адресная книга"

Адресная книга предназначеная для подбора и выбора получателей электронных писем при использовании встроенного почтового клиента.

Адресная книга может открываться в 2-х режимах – выбора и подбора.

В режиме подбора форма состоит из 2-х частей. Слева – источники подбора, справа – результат подбора.

Источники подбора:

  • Контрагенты, контактные лица контрагентов
  • Физические лица
  • Прочие контактные лица
  • Организации
  • Личные контакты
  • Пользователи
  • Группы объектов рассылки

Результат подбора – это табличные поля с получателями электронного письма (кому, копии и скрытые копии).

В режиме выбора элемента из адресной книги, результат подбора скрыт.

При навигации по табличным полям источников подбора, под каждым полем имеется поле выбора, значение которого меняется. В данном поле ввода отображаются доступные адреса электронной почты текущего объекта, по умолчанию подставляется тот e-mail, который является основным у объекта. Если адрес электронной почты всего один – то полю выбора устанавливается режим «Только чтение». Перед добавлением объекта в то или иное табличное поле результата подбора, пользователь может выбрать необходимый адрес электронной почты. При отсутствии какого либо адреса эл.почты, получатель не добавляется.

На закладке «Группы объектов» пользователь может добавить в получателей сразу несколько объектов, которые были предварительно объединены в группу объектов рассылки. Можно добавить как все объекты выбранной группы, так и выбрать только нескольких получателей.

На закладке «Поиск» пользователь может по части наименования или адреса электронной почты объекта (или нескольким частям объединенным по «И») найти всех получателей, и выбрать только необходимых. При этом поиск производится не только по первым значениям, а на вхождение искомого значения в наименование или e-mail объекта.

В табличных полях результата подбора, в колонках «E-mail» есть возможность изменить адрес уже подобранного объекта, выбрав его из выпадающего списка, содержащего все адреса электронной почты объекта.

По кнопке «ОК» нижней командной панели формы производится добавление подобранных объектов а соответствующие поля электронного письма.
