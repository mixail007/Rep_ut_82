Журнал документов "Документы по ОС"

Журнал предназначен для ввода, просмотра и редактирования документов по учету основных средств:

  • "Принятие к учету ОС";
  • "Изменение состояния ОС";
  • "Комплектация ОС";
  • "Модернизация ОС";
  • "Перемещение ОС";
  • "Подготовка к передаче ОС";
  • "Передача ОС";
  • "Списание ОС".
