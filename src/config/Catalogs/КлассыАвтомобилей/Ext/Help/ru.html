Справочник "Классы автомобилей"

Назначение

В справочник классов автомобилей заносится информация обо всех классах автомобилей, обслуживаемых на предприятии. Согласно класса автомобиля устанавливается та или иная стоимость работ.

Относится к подсистеме: «Автосервис».

Предопределенные элементы

  • Общие работы. Классы автомобилей. По умолчанию в справочнике существует предопределенный элемент "Общие работы", используемый для задания цен авторабот для всех классов автомобилей (если не указано противное). Удаление и редактирование данного элемента справочника недоступно. Класс автомобиля задается для всех моделей автомобилей, согласно чему и производиться расчет цен на выполнения тех или иных работ согласно заказ-наряду.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 100.

Не иерархический.

Формы

  • Форма списка. Основная форма списка, выбора и редактирования. Используется для отображения списка, выбора элементов и редактирования справочника.

Способ вызова

Вызывается из меню "Справочники --> Автомобили --> Классы автомобилей".
