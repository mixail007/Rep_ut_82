Справочник "Фильтры для электронных писем"

Справочник используется во востроенном почтовом клиенте и предназначен для хранения фильтров электронных писем - признаков и действий, которые будут совершатся с принятыми электронными письмами.

Для каждой учетной записи настраиваются свои фильтры. Изменение состава фильтров производится в форме элемента справочника "Учетные записи электронной почты".

В табличном поле "Условия фильтра", настраиваются условия, по которым будут отбираться электронные письма при получении. Условия можно накладывать на следующие поля электронных писем:

  • Содержание поля "Отправитель"
  • Содержание поля "Тема"
  • Содержание текста письма
  • Содержание поля "Кому"
  • Содержание поля "Копии"

Под табличным полем условий пользователь может указать, как должны соединяться установленные условия (имеет смыл при количестве условий более одного) для данного фильтра.

В табличном поле "Действия фильтра" пользователь указывает, какие действия необходимо производить над электронными письмами, соответствующими условиям текущего фильтра.
