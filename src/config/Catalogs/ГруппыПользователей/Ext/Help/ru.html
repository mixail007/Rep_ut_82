Справочник "Группы пользователей"

Справочник "Группы пользователей" используется при назначении доступа пользователей к конкретным объектам конфигурации.

Каждый пользователь может быть отнесен к определенной группе пользователей. Добавление пользователя в группу пользователей производится с помощью кнопки "Группа пользователей" в справочнике "Пользователи". При нажатии на эту кнопку открывается дополнительное диалоговое окно.

 С помощью кнопки "Добавить" пользователь добавляется в группу пользователей, а с помощью кнопки "Удалить" , пользователь удаляется из группы пользователей.

В справочнике "Группы пользователей" на закладке "Состав" показывается список тех пользователей, которые относятся к данной группе. На закладке "Объекты доступа" вводится информация обо всех областях данных (ролях пользователей) и права доступа к конкретным объектам.
