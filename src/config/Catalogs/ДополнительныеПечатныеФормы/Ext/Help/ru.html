Справочник «Дополнительные печатные формы»

Справочник содержит список зарегистрированных дополнительных печатных форм для документов.

Корректно зарегистрированная печатная форма будет выводиться в общем списке печатных форм документа, вызываемого по кнопке "Печать". При этом форма будет помещена в конец этого списка, отделенная разделителем.

 

Регистрация новой печатной формы выполняется в форме элемента данного справочника. При этом следует указать:

  • Наименование печатной формы - под этим именем данная формы будет добавлена в общий список печатных форм документа.
  • Внешний файл - файл внешней обработки формирования печатной формы.
  • Принадлежность печатной формы - список документов, к которым будет принадлежать данная форма.

Не все документы позволяют использовать дополнительные печатные формы

В форме списка возможно выполнение переноса всех зарегистрированных дополнительных печатных форм между различными базами. Вызов формы переноса осуществляется с помощью кнопки "Экспорт / импорт". В данной форме следует указать имя файла, в котором (из которого) будет сохранен (получен)  список форм. Сохранение списка в файле осуществляется нажатием кнопки "Экспорт", соответственно получение - "Импорт".

Требования к внешней обработке, регистрируемой в качестве дополнительной печатной формы.

Для возможности корректного использования внешней обработки в качестве дополнительной печатной формы, данная обработка должна соответствовать некоторым обязательным условиям:

  1. Одним из реквизитов внешней обработки должен быть реквизит с именем "СсылкаНаОбъект", с типом значения, содержащим ссылку на объект для которого данная формы будет зарегистрирована.
  2. В модуле внешней обработки должна быть определена экспортная функция "Печать()", возвращающая объект "Табличный документ" сформированной печатной формы. Показ данного табличного документа будет осуществляться средствами самого документа.

 

Кроме того, определены дополнительные условия для внешней обработки, позволяющие выполнять регистрацию дополнительной печатной формы полностью в автоматическом режиме:

  • В синониме внешней обработки допускается указывать пользовательское имя печатной формы. Это значение будет использовано при интерактивной регистрации формы.
  • Во внешней обработке может быть создан дополнительный  макет с именем "Параметры_Авторегистрации", в котором в первой колонке возможно перечисление объектов метаданных конфигурации, для которых данная форма должна быть использована. Например, "Документы.ПоступлениеТоваровУслуг" и т.д. Данные параметры будут использованы при интерактивной регистрации данной обработки.
