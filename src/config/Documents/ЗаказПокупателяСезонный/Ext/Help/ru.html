Документ "Заказ покупателя Сезонный"

Документ "Заказ покупателя Сезонный" предназначен для анализа и формирования заказов покупателя на сезон.

 

Если выбран Договор контрагента - Анализ и формирование заказов покупателя
            будет проводиться ТОЛЬКО с учетом этого договора.

 

[v] Транзит  -  позволяет анализировать остатки на складах выбранного Подразделения (филиала)

 

[v] Автокомплектация  -  автоматически (раз в час) - формирует Заказы покупателя
в соответствие с потребностью Клиента и наличию товаров в свободном остатке.

*** для дисков - только комплектами по 4 шт.


Один производитель  - Если поле заполнено, проверяются товары в табличной части на соответствие производителю.

 

[] Автокомплектация с учетом аналогов  - Комплектуется номенклатурой с учетом аналогов

[]  Учитывать аналоги - в случае снятия галки "Автокомплектация с учетом аналогов" выполнение сезонника все равно будет считаться с учетом отгруженных аналогов. Если галку снять, то выполнение будет считаться только по номенклатуре

[] По штамповке - доп. разрешение на комплектацию штамповки

[] по товарам в пути - признак, что заказ создан из терминала
