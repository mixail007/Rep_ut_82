Документ "Задание на транспорт"

Документ предназначен для учета отгрузок по машинам и создания документов от Перевозчика. 

Для оперативного анализа состояния заданий на транспорт -
 - в списке заданий принято определенное цветовое оформление:

Цвет текста - определяет наличие документов по заказу:

Обычный черный текст - неотгруженные задания,
Черный текст - неотгруженные, но редактированные за последние 2 часа ,
Синий текст - есть счета (поступления доп.расходов / услуг от перевозчика) по данному заданию.

Цвет фона строки - определяет состояние заказа:

задание полностью снято например

по заданию всё полностью отгружено  например
