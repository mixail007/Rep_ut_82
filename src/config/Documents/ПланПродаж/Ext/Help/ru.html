Документ "План продаж"

Документ предназначен для отражения в управленческом учете следующих операций:

  • Ввод в систему и хранение данных по предполагаемым объемам продаж
  • Формирование планов продаж по фактическим данным смежных подсистем и данных прошлых периодов

Ввод в систему и хранение данных по предполагаемым объемам продаж

В документе может указываться оценка предполагаемых объемов продаж для элементов номенклатуры (уточненное планирование) или для номенклатурных групп (укрупненное планирование). Оценка продаж указывается для отдельного периода по отдельному сценарию, при этом может производиться уточнение до подразделения и / или проекта.

Реквизиты «Сценарий», «Дата планирования» являются обязательными. Заполнение реквизита «Сценарий» определяет:

  • периодичность планирования, используемую в данном документе;
  • объект планирования – номенклатурные позиции или номенклатурные группы (ввод данных по одному сценарию планирования одновременно и по номенклатуре и по номенклатурным группам невозможен).

Реквизит «Дата планирования» выравнивается по периодичности планирования (на начальную границу).
Реквизиты «Проект» и «Подразделение» позволяют дополнительно отнести вводимые данные к какому-либо проекту или подразделению.

Состав плановых данных вводится в табличную часть документа. Номенклатурные позиции указываются с точностью до характеристики, если соответствующие параметры номенклатуры ведутся. По каждому объекту указывается единица измерения, плановый объем продаж в количественной (если по сценарию ведется планирование количества продаж) оценке и суммовой (если по сценарию ведется планирование сумм продаж) оценке с выделением НДС продаж.

Валюта документа, тип цен, признаки учета НДС настраиваются штатным для документов образом – в диалоговом окне «Цены и валюта» документа.

Документа может заполняться подбором из справочников «Номенклатура» или "Номенклатурные группы".

Формирование планов продаж по по данным подсистемы планирования и смежных подсистем

Состав документа может быть заполнен в автоматическом режиме. Для этого используется "Помощник планирования", вызываемый по кнопке «Заполнить».

 

Процент наценки и Валовая прибыль
В документе добавлена возможность ввода Среднего процента наценки для "Валовой прибыли",
ввода и распределения общей суммы Валовой прибыли, а также
ввода Валовой прибыли для каждой строки - в отдельности.

Примечание:
Валовая прибыль не может быть больше общей суммы, так как входит в ее состав.
Процент наценки расчитывается по формуле:
Процент = 100% * ВаловаяПрибыль / (СуммаДокумента - ВаловаяПрибыль);
