Документ "План закупок"

Документ предназначен для отражения в управленческом учете следующих операций:

  • Ввод в систему и хранение данных по предполагаемым объемам закупок
  • Формирование планов закупок по фактическим данным смежных подсистем и данных прошлых периодов

Ввод в систему и хранение данных по предполагаемым объемам закупок

В документе может указываться оценка предполагаемых объемов закупок для элементов номенклатуры (уточненное планирование) или для номенклатурных групп (укрупненное планирование). Оценка закупок указывается для отдельного периода по отдельному сценарию, при этом может производиться уточнение до подразделения и / или проекта.

Реквизиты «Сценарий», «Дата планирования» являются обязательными. Заполнение реквизита «Сценарий» определяет:

  • периодичность планирования, используемую в данном документе;
  • объект планирования – номенклатурные позиции или номенклатурные группы (ввод данных по одному сценарию планирования одновременно и по номенклатуре и по номенклатурным группам невозможен).

Реквизит «Дата планирования» выравнивается по периодичности планирования (на начальную границу).
Реквизиты «Проект» и «Подразделение» позволяют дополнительно отнести вводимые данные к какому-либо проекту или подразделению.

Состав плановых данных вводится в табличную часть документа. Номенклатурные позиции указываются с точностью до характеристики, если соответствующие параметры номенклатуры ведутся. По каждому объекту указывается единица измерения, плановый объем закупок в количественной (если по сценарию ведется планирование количества закупок) оценке и суммовой (если по сценарию ведется планирование сумм закупок) оценке с выделением НДС закупок.

Валюта документа, тип цен, признаки учета НДС настраиваются штатным для документов образом – в диалоговом окне «Цены и валюта» документа.

Документа может заполняться подбором из справочников «Номенклатура» или "Номенклатурные группы".

Формирование планов закупок по по данным подсистемы планирования и смежных подсистем

Состав документа может быть заполнен в автоматическом режиме. Для этого используется "Помощник планирования", вызываемый по кнопке «Заполнить».
