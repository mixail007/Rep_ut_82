Документ "Вычет НДС с аванса"

Документ предназначен для автоматизированного предъявления к вычету сумм налога на добавленную стоимость, ранее исчисленных с полученных авансов.

В связи с этим пользоваться документом рекомендуется только после того, как:

  • будут проверены расчеты с покупателями по авансам полученным;
  • будут зарегистрированы счета-фактуры на авансы полученные.

При наличии в отчетном периоде дебетового оборота по счетам учета авансов полученных и регистрации счетов-фактур на авансы предлагается принять НДС с авансов к вычету (соответствующая строка заносится в табличную часть документа).

Табличная часть документа доступна для редактирования.
