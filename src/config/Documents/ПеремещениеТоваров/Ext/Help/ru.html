Документ "Перемещение товаров"

Документ "Перемещение товаров" предназначен для оформления передачи номенклатурных позиций между  различными складами: оптовыми, розничными, неавтоматизированными торговыми точками.

В качестве склада "Отправитель" указывается склад, с которого производится отгрузка товара, а в качестве склада "Получатель" указывается тот склад, на который поступает товар.

В том случае, если товар перемещается в неавтоматизированную торговую точку, то необходимо в документе указать те розничные цены, по которым товар будет продаваться в неавтоматизированной торговой точке.

Цены, зафиксированные в данном документе, будут являться отпускными ценами, по которым будет производиться продажа и учет номенклатурных позиций в неавтоматизированной торговой точке.

Изменение розничных цен в неавтоматизированной торговой точке производится документом «Переоценка товаров в рознице».
