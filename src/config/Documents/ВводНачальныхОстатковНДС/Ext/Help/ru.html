Документ "Ввод начальных остатков по НДС"

Документ позволяет ввести первоначальные данные для работы подсистемы учета НДС.

Его следует использовать при начале эксплуатации программы.

Данный документ формирует специальные движения в регистрах подсистемы учета НДС. С его помощью можно ввести остатки по НДС, полученному от поставщиков, остатки авансов полученных от покупателей и остатки по НДС, который отгружен покупателям, но еще не оплачен ими.
