Отчет "Остатки товаров к получению"

В отчете "Остатки товаров на складах" можно вывести данные об остатках неотфактурованных товаров, получение которых оформлено с помощью документа "Приходный ордер на товары".

В отчете можно установить быстрый отбор по складам компании, по номенклатуре.

Если в настройках отчета установить "Зарезервированные" -равно- истина, то в отчете будут отображаться только те товары, которые зарезервированы по приходному ордеру (в документе "Приходный ордер на товары" установлен флаг "Без права продажи"). 

Количество товаров в отчете может выводиться в базовых единицах измерения, единицах хранения остатков и в единицах для отчетов.

Отчет может быть выведен в виде кросс-таблицы или в виде списка.
