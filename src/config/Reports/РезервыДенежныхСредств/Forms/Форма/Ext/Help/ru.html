Отчет "Резервы денежных средств"

Отчет предназначен для получения информации по исполнению заявок на расходование средств, суммы которых резервируются по местам хранения денежных средств.

Значения в ячейках отчета отражают остатки по заявкам, денежные средства под которые зарезервированы (исключены из доступного к использованию остатка), но расходные платежные документы по которым еще не оформлены.

Значения в ячейках отчета расшифровываются с помощью этого же отчета с выбором дополнительной детализации.
