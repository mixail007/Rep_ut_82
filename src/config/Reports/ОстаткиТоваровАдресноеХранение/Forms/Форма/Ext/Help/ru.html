Отчет "Остатки товаров на складах"

В отчете "Остатки товаров на складах" можно вывести данные об остатках товаров с распределением по складам.
 
В отчете можно установить быстрый отбор по складам компании, по номенклатуре.

Товары в отчете могут выводиться в базовых единицах, единицах хранения остатков товаров и единицах для отчетов. 

Отчет может быть выведен в виде кросс-таблицы или в виде списка. Для того , чтобы отчет выводился в виде кросс-таблицы необходимо в настройках отчета установить значения в группировках колонок и группировках строк. Если в список "Группировка колонок" не выбрано ни одного значения, то отчет выводится в виде списка.  

Для складов вида "Неавтоматизированная торговая точка" предусмотрена возможность оценки товаров в розничных ценах.
