Регистр сведений "Нормочасы работ"

Назначение

Предназначен для отражения ценовой политики компании на выполнения авторемонтных работ. В нем хранятся цены выполнения авторемонтных работ в разрезе классов автомобилей.

Относится к подсистеме: «Автосервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Авторабота. Ссылается на справочник «Автоработы». Авторабота, цена которой задается.
  • Класс автомобиля. Ссылается на справочник «Классы автомобилей». Класс автомобиля, для которого задается цена.

Ресурсы

  • Нормочас. Ссылается на справочник «Нормочасы». Нормочас выполнения автоработы.
  • Цена. Тип значения: «Число». Цена выполнения автоработы.

Способ вызова

Вызывается из справочника «Автоработы» или «Классы автомобилей» из меню "Перейти" на панели формы списка или элемента. Редактирование нормочасов контрагентов также возможно из формы списка справочника «Автоработы».
