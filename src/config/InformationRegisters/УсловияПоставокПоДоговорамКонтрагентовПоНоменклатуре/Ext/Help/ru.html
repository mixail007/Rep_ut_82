Регистр сведений "Условия поставок по договорам контрагентов (по номенклатуре)"

Регистр предназначен для хранения данных документов "Условия поставок по договорам контрагентов", указанных в табличной части документа.

На основании данных регистра производится анализ выполнения дополнительных условия по договорам контрагентов и строится отчет "Выполнение условий по договорам контрагентов", заполняются цены номенклатуры в соответствующих документах.
