Регистр сведений "Параметры амортизации (управленческий учет)"

Содержит информацию о параметрах амортизации управленческого учета.  Регистр используется для расчета амортизации по управленческому учету.

Регистр периодический, изменяется документами:

  • "Ввод начальных остатков ОС";
  • "Изменение параметров начисления амортизации ОС";
  • "Модернизация ОС";
  • "Передача ОС";
  • "Принятие к учету ОС".
