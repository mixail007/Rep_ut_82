Правила Ценообразования (Экспорт)

Набор правил ценообразования для Экспортных клиентов  - так же как и правила ценообразования клиентов (руб)
"работают" в порядке очередности (поле Приоритет), но  в поле "Скидка в USD (Абсолютная)"

Порядок цен:

1) Цены распродажи - это фиксированные цены (в USD)  с приоритотом -7000

2) Цены по правилам - ОБЯЗАТЕЛЬНО должно быть хотя бы одно правило!
Для удобства заполнения - достаточно нажать кнопку "Цены"  или A, B или С  для ввода фиксированного набора правил

Дополнительно (с 29.01.2016г.) добавлена кнопка [Скидка %]

3) для всех остальных дисков - действует "заглушка"  Диски - цена по типу "Базовая USD" с приоритетом 1001

4) шины и другие товары - расчитываются по Базовая цена / Курс USD  * 100/118  с приоритетом 2000

по договорам на Экспорт (при установленной валюте договора USD или EURO)
Применяется в "Заказах покупателя" и срабатывает ТОЛЬКО при ручном нажатии на кнопку "Цены клиента"
