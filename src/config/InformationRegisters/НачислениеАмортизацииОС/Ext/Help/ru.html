Регистр сведений "Начисление амортизации"

Содержит информацию о начислении амортизации по основным средствам для целей управленческого учета.

В регистре хранится информация  о необходимости начисления амортизации в определенные периоды.

Регистр периодический. При расчете амортизации используется значение флага начисления амортизации по объекту на начало месяца.

Изменяется следующими документами:

  • "Ввод начальных остатков ОС";
  • "Принятие к учету ОС";
  • "Изменение состояния ОС";
  • "Подготовка к передаче ОС";
  • "Передача ОС";
  • "Списание ОС".
