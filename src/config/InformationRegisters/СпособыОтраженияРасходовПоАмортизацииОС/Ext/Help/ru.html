Регистр сведений "Способы отражения расходов по амортизации основных средств"

В регистре хранятся данные о соответствии способов отражения расходов по амортизации основным средствам для управленческого учета.

Данные из регистра используются при расчете амортизации.

Регистр периодический, изменяется документами:

  • "Ввод начальных остатков ОС";
  • "Изменение способов отражения расходов по амортизации ОС";
  • "Принятие к учету ОС".
