Задачи с оповещением

Общие сведения

Задача с оповещением служит для фиксации в системе рабочих поручений либо персональных заданий пользователем программы (автором) и передачи ее для исполнения другому пользователю программы (исполнителю). В задаче указывается срок исполнения и приводится ее описание. Инициатором задачи может выступать пользователь, а в определенных случаях, задача может быть создана системой автоматически, руководствуясь принятыми алгоритмами автоматической постановки задач. Существующие алгоритмы могут быть модифицированы под нужды конкретного процесса исключительно программными средствами. Единственным стандартным алгоритмом автоматической постановки задач в текущей версии системы, является напоминание о дне рождения контактного лица.

В момент времени, определяемый как время напоминания, задача должна оповестить исполнителя об истечении срока, отведенного на исполнение данной задачи. Получив оповещение о задаче, пользователь-исполнитель должен выполнить поставленную задачу и поставить отметку об ее исполнении.

Совокупность задач, поставленных заданному пользователю, представляет собой распорядок действий пользователя, запланированных к исполнению в будущих и настоящем периодах, а также историю исполненных пользователем заданий за прошедшие периоды. Соответственно все задачи пользователя делятся на: выполненные и невыполненные (активные) задачи, а также задачи к исполнению (в которых данный пользователь является исполнителем) и задачи для контроля прочих исполнителей (в которых данный пользователь является автором).

Совокупность напоминаний о задачах, отображаемая заданному пользователю, представляет собой перечень активных задач к немедленному исполнению. 

 

Новые задачи

Ввод новой задачи можно произвести в форме вышеперечисленных документов и объектов, используя механизм  ввода на основании. Ввод новой задачи производится нажатием кнопки  (добавить) или клавиши [Insert] на клавиатуре.

Редактирование значений параметров новой задачи производится в полях объекта "Форма задачи".

 

В качестве связанного "Объекта" (выбор доступен только для полных прав) могут быть выбраны документы:

 «Заказ покупателя», «Заказ поставщику», «Внутренний заказ», «Счет на оплату поставщику», «Счет на оплату покупателю», «Событие».

или справочники: «Контрагенты», «Контактные лица», «Прочие контактные лица», «Договоры контрагентов».

Задачи текущего пользователя

Совокупность задач, в которых текущий пользователь определен как автор, либо как исполнитель, доступна для просмотра и совершения элементарных операций в объекте "Форма списка задач".

 

Напоминания о задачах

В запланированное время система оповещений будет выдавать пользователю соответствующее напоминание о задаче, для которой он установил флаг "Напомнить" и определил время напоминания. Для работы с напоминаниями о задачах, предусмотрена форма "Оповещение о задачах".

 

* Условия использования задач (включения/выключения)

 

 Для работы системы on-line напоминаний - необходимо, чтобы была включена настройка:  "Использовать Напоминания"

в Настройках пользователя(Исполнителя).

 

а также, должна быть заполнена константа: Интервал Проверки Напоминаний (в сек.) 
Если Интервал Проверки Напоминаний равен нулю, то система напоминаний - выключается для всех пользователей!

 

Доступ к списку задач:

Для просмотра списка задач: у пользователя (исполнителя) - должна быть включена настройка: "Показывать список задач при запуске" 

 А для пользователей с полными правами и менеджеров по закупкам - список задач можно открыть:

из главного меню в пункте: Сервис -> Задачи с оповещениями

   -> Список задач  - открывает список всех задач,
   -> Напоминания по задачам  -открывает сисок только тех задач, по которым установлены напоминания .
