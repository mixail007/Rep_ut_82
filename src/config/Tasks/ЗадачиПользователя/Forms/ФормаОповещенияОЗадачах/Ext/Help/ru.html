Задачи с оповещением. Форма оповещения о задачах

Общие сведения

Все задачи, требующие напоминания, сканируются периодически с заданной периодичностью проверки. Интервал проверки напоминаний в секундах задается в настройках параметров учета. В указанное время (время напоминания), с учетом поправки на периодичность проверки напоминаний, система оповестит пользователя о задаче, либо о нескольких задачах в едином списке формы оповещения о задачах. Данная форма представляет собой форму списка активных задач, время напоминания которых истекло. При этом, форма обладает определенным набором элементов, позволяющих выполнять ограниченное количество действий, необходимых для работы с активными задачами.

Открытая форма оповещения о задачах не может содержать пустой список активных задач.

Действия над задачами

Форма позволяет осуществить выбор задач из списка, выбрав как одну единственную, так и совокупность задач.

Каждая из выбранных задач может быть открыта в форме задачи, либо двойным щелчком по позиции задачи в списке, либо нажатием кнопки "Открыть".

Каждая из выбранных задач может быть отмечена как выполненная без открытия формы задачи,- для этого предусмотрена кнопка "Выполнено".

Каждая из выбранных задач может быть отложена по нажатию кнопки "Отложить": задача останется активной, а время напоминания такой задачи будет перенесено на срок, выбранный из предопределенных значений (5 минут, 10 минут, 30 минут, 1 час и т. д.) выпадающего списка в поле, слева от кнопки "Отложить".
